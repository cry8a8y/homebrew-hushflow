# Homebrew Tap for HushFlow

Turn AI thinking time into mindful breathing.

## Install

```bash
brew install cry8a8y/hushflow/hushflow
```

Then register hooks with your AI tools:

```bash
hushflow install
```

## Update

```bash
brew upgrade hushflow
```

## Uninstall

```bash
hushflow uninstall   # Remove hooks
brew uninstall hushflow
```
